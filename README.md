# АНСДИМАТ - Полевой калькулятор гидрогеолога

Мобильное приложение для гидрогеологических расчетов и анализа данных откачки скважин.

## 📱 Описание

АНСДИМАТ - это кроссплатформенное мобильное приложение, разработанное на React Native с использованием Expo. Приложение предназначено для гидрогеологов и специалистов, работающих с анализом данных откачки скважин.

## 🏗️ Архитектура проекта

### Основные компоненты

```
Ansdimat/
├── App.tsx                    # Главный компонент приложения
├── index.ts                   # Точка входа
├── app.json                   # Конфигурация Expo
├── eas.json                   # Конфигурация EAS Build
├── package.json               # Зависимости проекта
├── theme.js                   # Конфигурация тем
├── Localization.js            # Система интернационализации
├── LanguageContext.js         # Контекст языка
├── ThemeContext.js            # Контекст темы
├── components/                # Переиспользуемые компоненты
├── screens/                   # Экраны приложения
├── navigation/                # Навигация
├── utils/                     # Утилиты
└── assets/                    # Ресурсы (изображения, иконки)
```

### Структура модулей

#### Основные экраны
- **HomeScreen** - Главный экран с навигацией по разделам
- **CalculatorScreen** - Гидрогеологические расчеты
- **FieldDiaryScreen** - Полевой дневник с геолокацией
- **SettingsScreen** - Настройки приложения
- **AboutScreen** - Информация о приложении

#### Модуль обработки откачек (PumpingTestProcessing)
- **MainScreen** - Обзор проектов и быстрый доступ
- **NewWizard** - Мастер создания журналов
- **DataProcessing** - Обработка и анализ данных
- **JournalManager** - Управление журналами наблюдений
- **ProjectManager** - Управление проектами
- **ExportManager** - Экспорт результатов

## 📊 Функциональность

### Основные возможности
1. **Гидрогеологические расчеты** - Калькулятор для различных формул
2. **Обработка откачек** - Анализ данных откачки скважин
3. **Полевой дневник** - Запись данных с геолокацией
4. **Управление проектами** - Создание, импорт, экспорт проектов
5. **Визуализация данных** - Графики и диаграммы

### Технические особенности
- **Офлайн работа** - Все данные хранятся локально
- **Экспорт данных** - Поддержка JSON, CSV, PDF
- **Импорт проектов** - Загрузка существующих данных
- **Геолокация** - Запись координат точек наблюдения

## 🚀 Установка и запуск

### Предварительные требования
- Node.js 18+
- npm или yarn
- Expo CLI
- Android Studio / Xcode (для нативных сборок)

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm start
```

### Запуск на устройствах
```bash
# Android
npm run android

# iOS
npm run ios

# Web
npm run web
```

## 📦 Сборка приложения

### EAS Build
```bash
# Установка EAS CLI
npm install -g @expo/eas-cli

# Сборка для Android
eas build --platform android

# Сборка для iOS
eas build --platform ios
```

### Локальная сборка
```bash
# Android
expo run:android

# iOS
expo run:ios
```
