# Автоматическое определение системных настроек

## Обзор

Приложение теперь автоматически определяет и применяет системные настройки устройства для темы и языка.

## Функциональность

### Автоматическое определение темы

- **Системная тема**: Приложение автоматически наследует тему устройства (светлая/темная)
- **Сохранение настроек**: Пользовательские настройки темы сохраняются в AsyncStorage
- **Три режима**: Светлая, темная, системная (по умолчанию)

### Автоматическое определение языка

- **Системный язык**: Приложение определяет язык устройства и устанавливает соответствующий язык
- **Логика выбора**: 
  - Если системный язык русский → русский
  - Если любой другой язык → английский
- **Сохранение настроек**: Выбранный язык сохраняется в AsyncStorage

## Техническая реализация

### ThemeContext.js
- Использует `useColorScheme()` для определения системной темы
- Сохраняет настройки в AsyncStorage
- Поддерживает три режима: 'light', 'dark', 'system'

### LanguageContext.js
- Использует `expo-localization` для определения системного языка
- Автоматически устанавливает язык при первом запуске
- Сохраняет выбор пользователя в AsyncStorage
- Включает обработку ошибок и fallback механизмы
- Добавлена задержка инициализации для корректной загрузки

### Настройки в приложении
- В SettingsScreen тема отображается как кнопка вместо переключателя
- Показывается текущий режим темы (Светлая/Темная/Системная)
- Добавлен индикатор загрузки во время инициализации настроек

## Использование

### Для разработчиков

```javascript
// Получение текущей темы
const { themeMode, systemColorScheme } = useContext(ThemeContext);

// Получение текущего языка
const { locale, isLoading } = useContext(LanguageContext);

// Переключение темы (циклично: light → dark → system → light)
const { toggleTheme } = useContext(ThemeContext);

// Переключение языка (ru ↔ en)
const { toggleLanguage } = useContext(LanguageContext);
```

### Для пользователей

1. **Первый запуск**: Приложение автоматически определяет системные настройки
2. **Изменение темы**: Настройки → Внешний вид → Тема (циклическое переключение)
3. **Изменение языка**: Настройки → Язык → Изменить

## Совместимость
