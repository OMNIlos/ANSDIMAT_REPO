# Полевой дневник - Руководство пользователя

## Описание

Полевой дневник - это функциональный модуль мобильного приложения АНСДИМАТ, предназначенный для записи и управления полевыми данными с использованием карт и геолокации.

## Основные возможности

### 🗺️ Интерактивная карта
- Отображение текущего местоположения пользователя
- Гибридный тип карты (спутник + дороги)
- Масштабирование и перемещение по карте
- Компас и масштабная линейка

### 📍 Управление точками
- **Добавление точек** - создание точек с текущего местоположения
- **Редактирование** - изменение названия, описания и типа точек
- **Удаление** - удаление отдельных точек или всех сразу
- **Фильтрация** - отображение точек по типам

### 🏷️ Типы точек
1. **Наблюдение** (зеленый) - общие наблюдения в поле
2. **Проба** (синий) - места отбора проб
3. **Измерение** (оранжевый) - точки измерений
4. **Фото** (фиолетовый) - места фотографирования
5. **Заметка** (серый) - текстовые заметки

### 📊 Статистика
- Общее количество точек
- Количество точек за сегодня
- Количество точек за последние 24 часа
- Самый активный тип точек
- Детальная статистика по типам с процентами

### 💾 Сохранение данных
- Автоматическое сохранение в локальном хранилище
- Экспорт данных в CSV формат
- Возможность поделиться файлом

## Как использовать

### Добавление точки
1. Нажмите кнопку "+" (FAB) в правом нижнем углу
2. Введите название точки
3. Выберите тип точки
4. Добавьте описание (необязательно)
5. Нажмите "Добавить"

### Просмотр точек
1. Нажмите кнопку списка (📋) в правом верхнем углу
2. Используйте фильтры для отображения нужных типов точек
3. Нажмите на точку в списке для центрирования карты

### Редактирование точки
1. Нажмите на маркер на карте
2. В появившемся окне нажмите "Изменить"
3. Внесите изменения
4. Нажмите "Сохранить"

### Удаление точки
1. Нажмите на маркер на карте
2. В появившемся окне нажмите "Удалить"
3. Подтвердите удаление

### Просмотр статистики
1. Нажмите кнопку аналитики (📊) в правом верхнем углу
2. Просмотрите статистику по точкам

### Экспорт данных
1. Нажмите кнопку меню (⋮) в правом верхнем углу
2. Выберите "Экспорт данных"
3. Выберите способ отправки файла

## Управление картой

### Кнопки управления
- **📍 Моё местоположение** - центрирует карту на текущем местоположении
- **📋 Список точек** - показывает/скрывает список всех точек
- **📊 Статистика** - показывает/скрывает статистику
- **⋮ Меню** - дополнительные функции (экспорт, удаление всех точек)

### Навигация по карте
- **Масштабирование** - жесты щипка или двойной тап
- **Перемещение** - перетаскивание карты
- **Центрирование** - нажатие на точку в списке

## Разрешения

Приложение запрашивает следующие разрешения:
- **Местоположение** - для определения текущей позиции и добавления точек
- **Файловая система** - для экспорта данных

## Формат экспорта

Данные экспортируются в CSV формат со следующими полями:
- ID точки
- Название
- Тип
- Описание
- Широта
- Долгота
- Дата создания
- Точность (в метрах)

## Советы по использованию

1. **Регулярно экспортируйте данные** - для резервного копирования
2. **Используйте описания** - для детальной информации о точках
3. **Выбирайте правильные типы** - для лучшей организации данных
4. **Проверяйте точность GPS** - перед добавлением важных точек
5. **Используйте фильтры** - для работы с большим количеством точек

## Техническая поддержка

При возникновении проблем:
1. Проверьте разрешения на местоположение
2. Убедитесь в наличии интернет-соединения для загрузки карт
3. Перезапустите приложение при необходимости
